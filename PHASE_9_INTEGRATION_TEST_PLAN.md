### 3. 端到端流程测试

#### 3.1 完整项目工作流
```
1. 用户登录
2. 创建新项目
3. 查看项目列表（验证统计数据）
4. 添加样本到项目
5. 上传文件到样本
6. 创建分析任务
7. 监控任务执行（WebSocket）
8. 查看任务结果
9. 导出数据
10. 归档项目
```

#### 3.2 CSV导入工作流
```
1. 选择项目
2. 准备CSV文件
3. 导入样本
4. 验证样本数据
5. 检查关联关系
```

#### 3.3 权限验证流程
```
1. 普通用户登录
2. 尝试访问其他用户的项目（应失败）
3. 尝试访问自己的项目（应成功）
4. 管理员登录
5. 访问所有项目（应成功）
```
