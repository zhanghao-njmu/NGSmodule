### 数据流示例 - 任务执行
```
1. 用户点击"执行任务" → TaskList.tsx
2. taskStore.executeTask(taskId, config)
3. taskService.executeTask(taskId, data) → HTTP POST /api/v1/tasks/{id}/execute
4. 后端: tasks.py → Celery.delay(run_ngs_pipeline)
5. Celery Worker: 执行NGS脚本 → 更新progress → 发送WebSocket消息
6. WebSocket: server → client
7. websocketService 接收消息 → taskStore.handleWebSocketMessage
8. taskStore 更新状态 → React 重新渲染
9. UI: 进度条更新 + 状态标签更新 + 通知弹窗
```
